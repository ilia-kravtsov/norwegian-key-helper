# Norwegian Key Helper 🇳🇴

A simple Chrome extension that lets you quickly type Norwegian letters using keyboard shortcuts
if you have only English characters on your keyboard

## Features

- Alt + O → ø
- Alt + A → å
- Alt + E → æ
- Shift + Alt + O → Ø
- Shift + Alt + A → Å
- Shift + Alt + E → Æ

Works on Chrome:
- websites
- forms
- chats
- editors
- any text input and textarea fields

## Installation (from ZIP file)

Download this project as a ZIP archive:

### 1. Unzip the project
Extract the ZIP file to any folder on your computer

### 2. Open Chrome Extensions page
Open Google Chrome and insert to url:

chrome://extensions/

### 3. Enable Developer Mode
In the top-right corner, turn ON Developer mode

### 4. Load the extension
Click the button: "Load unpacked"

Then select the folder where you unzipped the project

### 5. Done
The extension is now installed and ready to use

## How to use

Just go to any text field on any website and use:

- `Alt + O` → ø
- `Alt + A` → å
- `Alt + E` → æ

Hold **Shift** for uppercase letters:

- `Shift + Alt + O` → Ø
- `Shift + Alt + A` → Å
- `Shift + Alt + E` → Æ

## Notes

- Works only when typing inside input fields or textareas.
- Some websites may override keyboard shortcuts.
